# Support

- Use [GitHub Issues](https://github.com/nawinds/poeditor/issues) for reproducible bugs and feature
  requests.
- Use `support@nawinds.dev` for private support and security-sensitive reports.
- Include the IDE version, plugin version, operating system, and a minimal sample catalog.

Questions about GNU gettext itself are best answered by the
[GNU gettext documentation](https://www.gnu.org/software/gettext/manual/).
